User

Define a classe referente aos usuários da API

class src.libs.user.User(name: str, email: str, password: str, roles: list[int])

Contém as informações de um usuário

Parâmetros

  • name (str) – Nome do usuário

  • email (str) – E-mail do usuário

  • password (str) – Senha de acesso do usuário na API

  • roles (list[int]) – Lista de identificadores das permissões do usuário na API

grant_role(role_identifier: int) → None

Adiciona uma permissão específica, se esta já não existe na instância

Parâmetros

role_identifier (int) – Identificador da permissão

revoke_role(role_identifier: int) → None

Remove uma permissão específica

Parâmetros

role_identifier (int) – Identificador da permissão
